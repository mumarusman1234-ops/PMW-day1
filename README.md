## 📌 Project Overview
Welcome to the core repository for **PMW-Day1**. This project serves as a clean, responsive personal landing page designed to showcase web development foundations, projects, and social links. 

> "The best way to predict the future is to invent it." — Alan Kay

### ✨ Features
* 📱 **Fully Responsive Layout** – Optimized for mobile, tablet, and desktop viewing.
* 🎨 **Modern Glassmorphism UI** – Clean aesthetics with subtle shadows and blurs.
* ⚡ **Fast Loading** – Built with pure, lightweight semantic HTML and vanilla CSS.

To run this project locally, simply clone the repository and open the index file:

```bash

# Navigate into the project folder
cd PMW-day1

# Open index.html in your default browser
open index.html
